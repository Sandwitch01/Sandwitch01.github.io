个人博客的搭建

此文章参考b站codesheep的视频，感谢。

1.工具：
node.js(含npm包管理器)
Git(相当于Linux终端窗口)
2.安装hexo:
新建一个博客文件夹,例：blog
在blog文件夹下，鼠标右键打开git bash here
node -v
npm -v
查看node和npm是否安装好

因为npm比较慢，所以这里安装cnpm来提高速度
npm install -g cnpm --registry==https://registry.npm.taobao.org
运行完后敲cnpm查看是否安装好，安好后就可以用cnpm来安装hexo
cnpm install -g hexo-cli
hexo -v   验证

3.开始使用hexo搭建博客
hexo init 初始化博客
hexo s 启动博客(本地端口localhost:4000)
hexo n “博客标题” 新建一篇博客

hexo g                 生成博客
hexo s                 可以看到新建的博客已经在localhost:4000

4.部署到github
新建一个仓库 注意：仓库名要和用户名一样（命名格式：用户名.github.io）
在blog文件夹中安装git部署的插件
cnpm install –save hexo-deployer-git
设置_config.yml:
deploy:
type: git
repo: github的仓库地址
branch: master
##注意冒号后面有空格

hexo d          开始部署到github
之后用用户名.github.io即可访问自己部署的hexo博客

博客主题的替换

1.下载主题
git clone 主题地址.git themes/文件夹名
例：git clone https:github.com/litten/hexo-themes-yilia.git themes/yilia
2.替换
先修改blog下的_config.yml中的theme theme:yilia
hexo clean 清理一下
hexo g 重新生成
hexo s 在本地先看看
hexo d 部署到github

(pwd：查看当前位置)一定要在博客文件夹下
(部署时碰到deploy not found：npm install –save hexo-deployer-git即可)
